# ReviewGuard AI

An NLP-powered Fake Product Review Detection System built using Machine Learning and Django.

## Features

- Fake Review Detection
- Trust Score Generation
- Risk Level Classification
- Explainable AI
- Review History Storage
- Django Web Interface

## Tech Stack

- Python
- Scikit-Learn
- TF-IDF
- Logistic Regression
- Django
- SQLite
- Bootstrap

## Accuracy

88% Classification Accuracy
